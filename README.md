# shortlink
Simple shortlink website built with NodeJS<br>
Installation :<br>
```bash
apt install update
apt upgrade -y
apt install git nodejs-lts nano -y
npm i
```
<br><br>
create and edit .env files<br >
```bash
nano .env
```
.env :<br>
```env
URI=<MongoDB uri>
email=<google email>
token=<google token>
admin=<email to access /admin>
```
